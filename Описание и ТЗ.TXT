"BrainLock"

Автор: Никулин Владислав

Описание идеи:
"BrainLock" - это сайт, на котором людям будут доступны различные математические и логические задачи.
Чтобы получить полный перечень задач пользователю необходимо зарегестрироватся/войти в систему.

Техническое задание. 
Цель: создать сайт для презентации и проверки математических и логических задач, решенных пользователем.

Подробное описание приложения:
  ----------------------------------------------------------------------------------------------
    • Структура проекта
    
      1) main.py - файл, в котором описана вся логика всех страниц сайта
      
      2) Папка templates:
          • base.html - базовый шаблон, который наследуют все остальные страницы сайта
          • home.html - шаблон страницы с неполным перечнем задач
          • login.html - шаблон страницы авторизации
          • reg.html - шаблон страницы регистрации
          • task.html - шаблон страницы с задачами
   
      3) Папка static:
          Папка css:
            • main.css - стилевой файл для базового шаблона
          Папка image:
            (картинки)
              
      4) Папка forms:
          • user_forms.py - файл с программным описанием форм, относящихся к пользователю
            (авторизации и регистрации)
            
      5) Папка data:
          • __all_models.py - сборник моделей для работы с БД
          • done.py - файл, описывающий нумерацию решенных задач в БД
          • users.py - файл, описывающий таблицу с информацией о пользователях в БД
          • db_session.py - файл, используемый для подключения к БД и работы с ней
          
      6) Папка db:
          • DataBase.db - БД с информацией о пользователях и опубликованных аудиофайлах
          
  ----------------------------------------------------------------------------------------------
    • Проект состоит из нескольких страниц: главной (где количество задач и их проверка 
      зависят от регистрации пользователя), регистрационной(регистрация/вход) и страницы 
      с выбранным заданием
    • На всех страницах сверху расположены кнопки для авторизации или регистрации
      пользователя и кнопка для возвращения на главную страницу
    • Чтобы полноценно работать с сайтом, пользователь должен зарегистрироваться и
      авторизовываться при последующих посещениях.
    • Как только пользователь будет авторизован, текущая страница будет сменена на главную, в
      которой и заключается весь основной функционал
    • Главная страница представляет собой небольшое описание сайта и сам перечень задач 2х типов:
      на автоматической и самостоятельной проверке.
  ----------------------------------------------------------------------------------------------
